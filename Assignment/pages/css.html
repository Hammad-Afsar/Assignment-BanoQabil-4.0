CSS (Cascading Style Sheets)

CSS (Cascading Style Sheets) is a stylesheet language used to control the presentation, layout, and design of HTML documents. It allows web developers to style web pages by specifying colors, fonts, spacing, positioning, and animations.

Key Features of CSS

  1. Separation of Content & Design –

    CSS keeps styling separate from HTML structure, making code cleaner.

  2. Selectors & Properties –

    Styles are applied using selectors (element, class, ID) and properties (color, font-size, margin).

  3. Cascading & Inheritance –

    Styles are applied based on a priority system (inline, internal, external).

  4. Responsive Design –

    CSS enables web pages to adapt to different screen sizes using media queries.

  5. Animations & Transitions –

    CSS can create animations without JavaScript using @keyframes and transition.

Ways to Apply CSS

  1. Inline CSS (applies styles directly within an element)

  2. Internal CSS (inside style> in the HTML head>)

  3. External CSS (recommended, linked via a separate .css file)

CSS Selectors & Properties

  1. Basic Selectors

    • Element Selector (p {}) – Targets all < p> elements.
    • Class Selector (.class {}) – Targets elements with a specific class.
    • ID Selector (#id {}) – Targets a unique element.

  2. Box Model

    Controls spacing, including:

    • Margin – Space outside the element.
    • Border – Edge of the element.
    • Padding – Space inside the element.
